# Linux Distros Guide

A beginner-friendly website that helps people learn about Linux, compare popular distributions, and get started with the right setup.

## Live Website

Visit the site here: [Linux Distros Guide](https://Zkboiz.github.io/linux-distros-guide/)

## About This Project

Linux Distros Guide is a school project built to make Linux easier to understand for beginners.  
The website explains what Linux is, highlights popular distributions, compares their features, and gives users a simple path to getting started.

The goal is to create a clean, professional, and easy-to-use educational website that feels approachable for new users.

## Features

- Clear introduction to Linux.
- Beginner-friendly distribution overviews.
- Simple comparison sections.
- Getting started pages for individual distros.
- A suggestion box for feedback and recommendations.
- Responsive design for desktop and mobile.

## Built With

- HTML
- CSS
- JavaScript

## Project Structure

- `index.html` — Homepage
- `what-is-linux.html` — Linux introduction page
- `distributions.html` — Popular distributions
- `compare.html` — Distro comparison page
- `getting-started.html` — Getting started overview
- `suggestions.html` — Feedback page
- `about.html` — About the project
- `css/style.css` — Main stylesheet
- `js/nav.js` — Navigation menu script

## Getting Started

If you want to run the project locally:

1. Clone the repository.
2. Open the folder in VS Code.
3. Open `index.html` in your browser, or use a local server extension.
