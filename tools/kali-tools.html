Kali Linux Tools Overview

Kali Linux comes pre-installed with hundreds of penetration testing and security auditing tools. Below is an overview of the main tool categories and their purposes. These tools are powerful and should only be used ethically and legally with proper authorization.

Information Gathering

Tools for collecting passive information about targets, networks, and systems.

Nmap

Network exploration tool and security scanner. Maps networks, identifies active hosts, and discovers open ports.

Network Scanning
Whois

Retrieves domain registration information and DNS data to identify target ownership and organization details.

Domain Lookup
Shodan

Search engine for finding exposed devices and services on the internet using passive reconnaissance.

Search Engine
TheHarvester

Gathers information from public sources like search engines and DNS records about targets.

Email & Host Discovery
Maltego

Visual intelligence tool for mapping relationships between people, organizations, and infrastructure.

OSINT

Vulnerability Analysis

Tools for identifying weaknesses and potential security issues in systems and applications.

OpenVAS

Open Vulnerability Assessment System. Scans for known vulnerabilities in systems and applications.

Vulnerability Scanner
Nikto

Web server scanner that detects dangerous files, outdated software, and configuration issues.

Web Server
Lynis

Security auditing tool that performs comprehensive system hardening and security checks on Unix/Linux systems.

Linux Auditing
SQLmap

Automated SQL injection testing tool that detects and exploits SQL database vulnerabilities.

Database Security

Web Application Testing

Specialized tools for testing web applications and services for security vulnerabilities.

Burp Suite

Comprehensive platform for web application security testing including scanning, crawling, and manual testing.

Web Security
OWASP ZAP

Open Web Application Security Project tool for finding security vulnerabilities in web applications.

Web Application
Wfuzz

Web fuzzer for discovering hidden resources, parameters, and potential vulnerabilities through brute-forcing.

Fuzzing

Wireless Testing

Tools for analyzing and testing wireless networks and protocols.

Aircrack-ng

Complete suite for wireless network auditing including packet capture, cracking, and testing tools.

WiFi Security
Wireshark

Network protocol analyzer for capturing and inspecting network traffic in real-time.

Network Analysis
Kismet

Wireless network detector and sniffer that identifies wireless networks and devices.

Wireless Detection

Exploitation Tools

Advanced tools for testing security by attempting controlled exploitation of vulnerabilities.

Metasploit Framework

Comprehensive penetration testing framework with thousands of exploits, payloads, and auxiliary modules.

Exploitation
Exploit-DB

Database of public exploits and vulnerability proofs-of-concept for testing known security issues.

Exploit Database
BeEF

Browser Exploitation Framework for testing browser vulnerabilities and client-side security.

Browser Security

Sniffing & Spoofing

Tools for analyzing network traffic and testing network protocol vulnerabilities.

tcpdump

Command-line network packet capture and analysis tool for monitoring network traffic.

Packet Capture
Ettercap

Network security tool for man-in-the-middle attacks, network sniffing, and protocol analysis in test environments.

MITM Testing
Arpspoof

Network utility for testing ARP spoofing vulnerabilities on local networks.

Network Testing

Post Exploitation

Tools for maintaining access and gathering information after initial compromise (in authorized testing scenarios).

Meterpreter

Advanced payload in Metasploit Framework for command and control after successful exploitation.

Command & Control
Mimikatz

Credential dumping tool for extracting authentication tokens and passwords from Windows systems.

Credential Testing
Empire

Post-exploitation framework with agent-based command and control capabilities for testing persistence.

Persistence Testing

Forensics

Tools for analyzing systems, recovering data, and investigating security incidents.

Volatility

Memory forensics framework for analyzing RAM dumps and investigating live system memory.

Memory Analysis
Autopsy

Digital forensics platform for analyzing hard drives and file systems in forensic investigations.

Disk Forensics
Sleuth Kit

Toolkit and library for digital forensic analysis of file systems and disk images.

File System Analysis
⚠️ Responsible Use: All tools listed here are powerful and should only be used in authorized security testing scenarios. Unauthorized access to computer systems is illegal. Always:
  • Obtain written permission before testing any system you don't own
  • Understand and follow local laws regarding cybersecurity
  • Document all testing activities thoroughly
  • Maintain confidentiality of vulnerabilities and test results
  • Report findings responsibly to system owners
