Setting Up Fedora

Complete installation guide for Fedora Linux

Home > Installation > Fedora Setup

System Requirements

Fedora is efficient and works well on modern hardware with cutting-edge features:

Processor

Dual-core processor, 2GHz or faster

RAM

2GB minimum, 4GB recommended

Storage

20GB disk space minimum

Display

1024x768 resolution minimum

Why Fedora? Fedora is sponsored by Red Hat and focuses on innovation with latest technologies. It's ideal for users interested in development, system administration, or exploring modern Linux features.

Downloading Fedora

  1. Visit the official Fedora website: getfedora.org
  2. Click on "Workstation" for desktop users
  3. Select the 64-bit version (or 32-bit if you have older hardware)
  4. Save the ISO file to your computer (approximately 1.7GB)
  5. You'll need a USB drive with at least 4GB capacity
Fedora Editions: Fedora offers Workstation (for developers), Server (for servers), and other specialized editions. Workstation is recommended for beginners.

Creating Installation Media

You need to create a bootable USB drive from the ISO file:

Windows Users:

  1. Download Etcher or Rufus (free tools)
  2. Insert your USB drive
  3. Open the tool and select your Fedora ISO file
  4. Select your USB drive as the target
  5. Click "Flash" or "Write" and wait for completion

Mac Users:

  1. Download Etcher from balena.io/etcher
  2. Insert your USB drive
  3. Open Etcher and select your Fedora ISO
  4. Select the USB drive
  5. Click "Flash" and wait for completion

Linux Users:

  1. Use the terminal command: sudo dd if=fedora.iso of=/dev/sdX bs=4M
  2. Replace sdX with your USB drive identifier (check with lsblk)
  3. Wait for the operation to complete

Installation Process

Step 1: Boot from USB

  1. Insert the bootable USB drive into your computer
  2. Restart your computer
  3. Press the boot key during startup (usually F12, ESC, or DEL)
  4. Select the USB drive from the boot menu

Step 2: Fedora Live Environment

  1. Fedora will load from the USB drive
  2. You'll see the desktop with an "Install to Hard Drive" option
  3. Click "Install to Hard Drive" to begin installation

Step 3: Select Language and Time Zone

  1. Choose your language from the available options
  2. Click "Continue" to proceed to configuration
  3. Select your time zone
  4. Choose your keyboard layout

Step 4: Storage Configuration

  1. Select the disk where you want to install Fedora
  2. Choose automatic partitioning for first-time users
  3. For advanced users, manual partitioning is available
  4. Click "Done" to confirm

Step 5: Network and Hostname

  1. Connect to your network (Wi-Fi or Ethernet)
  2. Set your computer's hostname
  3. Click "Done"

Step 6: User Setup

  1. Create a user account with your name
  2. Set a username (lowercase, no spaces)
  3. Create a password (remember this!)
  4. Choose whether to make this user an administrator
  5. Click "Done"

Step 7: Begin Installation

  1. Review your settings
  2. Click "Begin Installation" to start
  3. Wait for the installation to complete (usually 10-20 minutes)
  4. You may be prompted to set a root password (optional for new users)

Step 8: Reboot

  1. Installation will complete
  2. Click "Finish Configuration"
  3. Remove the USB drive
  4. Click "Reboot" to start your new Fedora system

First Boot and Initial Setup

After restarting, you'll see the Fedora login screen:

  1. Select your user account
  2. Enter your password
  3. Complete the welcome setup wizard (optional)
  4. You're now ready to use Fedora!

Essential First Steps

1. Update Your System

Keep Fedora secure and up-to-date:

  1. Open the Software application
  2. Look for any available updates
  3. Install updates (you may need to enter your password)
  4. Restart if prompted

2. Enable Additional Repositories

Fedora has RPMFusion repositories with additional software:

  1. Search for "Software" in applications
  2. Look for repositories section
  3. Enable RPMFusion Free and RPMFusion Nonfree repositories

3. Install Development Tools

If you're interested in programming, install development tools using the Software application.

Troubleshooting

Computer Won't Boot from USB

  • Ensure Secure Boot is disabled in BIOS
  • Check that you're pressing the correct boot key
  • Verify the USB drive was properly created
  • Try a different USB port

Installation Fails

  • Ensure you have at least 20GB of free space
  • Try disabling Secure Boot in BIOS
  • Verify the ISO file wasn't corrupted during download

Network Issues After Installation

  • Open Settings and check network configuration
  • Try reconnecting to your network
  • Restart the networking service

Need Help?

  • Fedora Project Documentation: docs.fedoraproject.org
  • Fedora Community Forum: forums.fedoraproject.org
  • Ask for help in Linux forums and communities

You've successfully installed Fedora. Welcome to the cutting-edge Linux experience!
