Setting Up Debian

Complete installation guide for Debian Linux

Home > Installation > Debian Setup

About Debian

Debian is one of the oldest and most stable Linux distributions. Known for its stability, reliability, and massive software repository, Debian is an excellent choice for servers and users who prioritize stability over cutting-edge features. This guide will walk you through the installation process.

System Requirements

Debian is very flexible and runs on minimal hardware. Here are the recommended specifications:

Processor

Pentium or newer (even very old systems work)

RAM

512MB minimum, 2GB recommended

Storage

5GB disk space minimum, 20GB recommended

Display

Any graphics card (800x600 minimum)

Why Debian? Debian runs excellently even on old hardware. Many users repurpose aging computers with Debian. Its largest software repository ensures you'll find tools for almost any purpose.

Downloading Debian

  1. Visit the official Debian website: debian.org
  2. Click "Download" and select the current stable release
  3. Choose the appropriate architecture:
    • amd64 (64-bit, recommended for modern computers)
    • i386 (32-bit, for older systems)
  4. Download the netinst or DVD ISO file (netinst is smaller, ~400MB)
  5. You'll need a USB drive with at least 1GB capacity
netinst vs DVD: The netinst image is smaller and downloads packages during installation. The DVD includes commonly-used packages offline. For beginners with internet access, netinst is recommended.

Creating Installation Media

Create a bootable USB drive from the ISO file:

Windows Users:

  1. Download Etcher from balena.io/etcher or use Rufus
  2. Insert your USB drive
  3. Open the tool and select your Debian ISO file
  4. Select your USB drive as the target
  5. Click "Flash" or "Write" and wait for completion

Mac Users:

  1. Download Etcher from balena.io/etcher
  2. Insert your USB drive
  3. Open Etcher and select your Debian ISO
  4. Select the USB drive and click "Flash"
  5. Wait for completion

Linux Users:

  1. Use: sudo dd if=debian.iso of=/dev/sdX bs=4M
  2. Replace sdX with your USB drive (check with lsblk)
  3. Run: sudo sync to ensure all data is written

Installation Process

Step 1: Boot from USB

  1. Insert the bootable USB drive
  2. Restart your computer
  3. Press the boot key during startup (F12, ESC, or DEL)
  4. Select the USB drive from the boot menu

Step 2: Debian Installer Menu

  1. The Debian installer will load
  2. Select "Graphical Install" for the easiest installation
  3. The installer will begin hardware detection

Step 3: Choose Language

  1. Select your language (English recommended for beginners)
  2. Select your country/region
  3. Select your keyboard layout
  4. Click "Continue"

Step 4: Hostname and Domain

  1. Enter a hostname for your system (e.g., "my-debian")
  2. Leave the domain name blank (unless on a corporate network)
  3. Click "Continue"

Step 5: Root Password Setup

  1. Enter a root password (strong password recommended)
  2. Confirm the root password
  3. This password grants full system access, so keep it secure
  4. Click "Continue"

Step 6: Create User Account

  1. Enter your full name
  2. Enter a username (for daily use, not root)
  3. Create a strong password for your user account
  4. Click "Continue"

Step 7: Partition Disks

  1. Choose "Guided - use entire disk" for standard installation
  2. Select the disk where you want to install Debian
  3. Choose partitioning scheme (default is recommended for beginners)
  4. Review the summary and confirm the changes

Step 8: Installation

  1. The installer will now install the base system
  2. This may take 10-20 minutes depending on your hardware
  3. The system will download and configure packages
  4. Do not interrupt this process

Step 9: Software Selection

  1. You'll be asked which software to install
  2. For beginners, select "Debian desktop environment" and "GNOME"
  3. This provides a user-friendly graphical interface
  4. Click "Continue"

Step 10: Bootloader Installation

  1. The installer will ask about GRUB bootloader installation
  2. Answer "Yes" to install GRUB to the main disk
  3. Select /dev/sda as the device (usually the correct choice)
  4. Click "Continue"

Step 11: Installation Complete

  1. You'll see a completion message
  2. Click "Continue" to reboot into your new Debian system
  3. Remove the USB drive when prompted

First Boot and Initial Setup

After restarting, you'll see the Debian login screen:

  1. Enter your username and password (the regular user account you created)
  2. The desktop environment will load
  3. You now have a complete Debian system!

Essential First Steps

1. Update Your System

Keep Debian secure with the latest updates:

  1. Open a terminal
  2. Run: sudo apt update && sudo apt upgrade
  3. Enter your password if prompted
  4. Wait for updates to complete

2. Install Additional Software

Debian includes many applications, but you may want to add more:

  1. Open "Software" or "GNOME Software" application
  2. Search for applications you need
  3. Or use terminal: sudo apt install [package-name]

3. Configure System Settings

Customize your Debian installation:

  1. Open Settings application
  2. Configure display, keyboard, mouse settings
  3. Set up network connections if needed

4. Regular Maintenance

Keep Debian running smoothly:

  1. Run updates regularly (weekly or monthly)
  2. Keep your software current
  3. Backup important data regularly

Troubleshooting

Installation Won't Start from USB

  • Try pressing different boot keys (F12, ESC, DEL, F2)
  • Verify the USB was properly created
  • Try a different USB port or USB drive
  • Check if BIOS/UEFI settings need adjustment

Installation Freezes

  • Don't interrupt; installation takes time on slower systems
  • Ensure computer has adequate power supply
  • Try again with a different USB drive
  • Check for hardware issues (RAM, disk)

Can't Log In After Installation

  • Verify you're using the correct username and password
  • Caps Lock may be on - check and try again
  • If you forget the password, you may need to reinstall

Network Not Working

  • During installation, select "Configure the network automatically"
  • After installation, use Settings to configure network
  • For WiFi, you may need additional drivers
  • Check Debian documentation for your network hardware

Need Help?

  • Debian Official Documentation: debian.org/doc
  • Debian Wiki: wiki.debian.org
  • Debian Forums: forums.debian.net
  • Community support: Ask in Linux communities and forums

You've successfully installed Debian. Enjoy a stable, reliable operating system!
