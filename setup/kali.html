Setting Up Kali Linux

Complete installation guide for Kali Linux

Home > Installation > Kali Linux Setup
⚠️ Important Note: Kali Linux is designed for professional penetration testing and security research. It includes advanced security tools that should only be used ethically and legally on systems you own or have permission to test. Always follow local laws and ethical guidelines.

System Requirements

Kali Linux can run on various hardware. Here are the recommended specifications for a smooth experience:

Processor

Dual-core processor, 2GHz or faster

RAM

2GB minimum, 4GB recommended

Storage

20GB disk space minimum (more for tools)

Display

1024x768 resolution minimum

Downloading Kali Linux

  1. Visit the official Kali Linux website: kali.org
  2. Navigate to the Downloads section
  3. Select the appropriate image for your architecture (64-bit recommended for most users)
  4. Download the ISO file (approximately 3-4GB)
  5. You'll need a USB drive with at least 4GB capacity
Tip: Verify the ISO file's integrity using the provided SHA256 checksums before creating installation media. This ensures you have an authentic, unmodified download.

Creating Installation Media

Create a bootable USB drive from the ISO file:

Windows Users:

  1. Download Etcher from balena.io/etcher or use Rufus
  2. Insert your USB drive (at least 4GB)
  3. Open the tool and select your Kali Linux ISO file
  4. Select your USB drive as the target
  5. Click "Flash" or "Write" and wait for completion

Mac Users:

  1. Download Etcher from balena.io/etcher
  2. Insert your USB drive
  3. Open Etcher and select your Kali Linux ISO
  4. Select the USB drive
  5. Click "Flash" and wait for completion

Linux Users:

  1. Use Etcher or the terminal command: sudo dd if=kali.iso of=/dev/sdX bs=4M
  2. Replace sdX with your USB drive identifier (check with lsblk)
  3. Wait for the operation to complete

Installation Process

Step 1: Boot from USB

  1. Insert the bootable USB drive into your computer
  2. Restart your computer
  3. Press the boot key during startup (usually F12, ESC, or DEL—check your computer's manual)
  4. Select the USB drive from the boot menu

Step 2: Kali Linux Installer

  1. The Kali Linux installer will boot
  2. Select "Graphical Install" for a user-friendly installation
  3. Choose your language and location

Step 3: Choose Language and Keyboard

  1. Select your language (English is recommended)
  2. Select your keyboard layout
  3. Click "Continue"

Step 4: Hostname and Domain

  1. Enter a hostname for your system (e.g., "kali-pc")
  2. Leave the domain name blank unless you're on a corporate network
  3. Click "Continue"

Step 5: User Account Setup

  1. Enter your full name
  2. Enter a username for logging in
  3. Create a strong password (required for Kali)
  4. Click "Continue"

Step 6: Partition Disks

  1. Choose "Guided - use entire disk" for a standard installation
  2. Select the disk where you want to install Kali
  3. Choose a partitioning scheme (default is recommended)
  4. Review and confirm the partitioning changes

Step 7: Installation Completion

  1. Wait for the installation to complete (usually 10-20 minutes)
  2. The installer will install the base system and selected packages
  3. Click "Continue" when prompted
  4. Click "Restart" to reboot into your new Kali Linux system
  5. Remove the USB drive when prompted

First Boot and Initial Setup

After restarting, you'll see the Kali Linux login screen:

  1. Enter your username and password from installation
  2. Once logged in, you'll see the Kali desktop environment
  3. Open a terminal to check your system and update packages
  4. You're now ready to use Kali Linux!

Essential First Steps

1. Update Your System

Keep Kali Linux updated with the latest packages and security patches:

  1. Open a terminal
  2. Run: sudo apt update && sudo apt upgrade
  3. Wait for the updates to complete

2. Review Available Tools

Explore the tools available in Kali Linux:

  1. Open the Applications menu
  2. Browse the Kali Tools section to see what's available
  3. Visit our Kali Linux Tools Guide for detailed descriptions

3. Learn Best Practices

Understand ethical and legal use:

  1. Only use Kali tools on systems you own or have written permission to test
  2. Follow all applicable laws and regulations in your jurisdiction
  3. Keep detailed documentation of any security testing you perform
  4. Always get proper authorization before conducting penetration tests

4. Customize Your Setup

Install additional tools or software as needed for your specific use case.

Troubleshooting

Computer Won't Boot from USB

  • Check that you're pressing the correct boot key for your computer
  • Verify the USB drive was properly created with the ISO
  • Try a different USB port
  • Check if Secure Boot is enabled in BIOS and disable it if necessary

Installation Freezes or Hangs

  • Don't interrupt the process; installation can take time
  • Try disabling Secure Boot in BIOS settings
  • If it persists, try a different USB drive or computer
  • Check the Kali Linux forums for specific hardware issues

Login Issues

  • Ensure you're using the correct username and password from installation
  • Check that Caps Lock is not accidentally enabled
  • If you forget your password, you may need to reinstall

Need Help?

  • Kali Linux Official Documentation: docs.kali.org
  • Kali Linux Forums: forums.kali.org
  • Community Support: Ask in Linux forums and communities

Congratulations!
