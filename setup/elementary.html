Setting Up Elementary OS

Complete installation guide for Elementary OS

Home > Installation > Elementary OS Setup

System Requirements

Elementary OS is optimized for performance while maintaining beautiful design:

Processor

Dual-core processor, 1.5GHz or faster

RAM

2GB minimum, 4GB recommended

Storage

20GB disk space minimum

Display

1024x768 resolution minimum

Why Elementary OS? Elementary OS is based on Ubuntu and focuses on beautiful, intuitive design. It's perfect for users who want a visually elegant Linux experience without the learning curve.

Downloading Elementary OS

  1. Visit the official Elementary OS website: elementary.io
  2. Click on "Download"
  3. Choose the latest stable release
  4. Save the ISO file to your computer (approximately 2GB)
  5. You'll need a USB drive with at least 4GB capacity
Free or Pay? Elementary OS is free to download, but you can choose to contribute financially to support development.

Creating Installation Media

You need to create a bootable USB drive from the ISO file:

Windows Users:

  1. Download Etcher or Rufus (free tools)
  2. Insert your USB drive
  3. Open the tool and select your Elementary OS ISO file
  4. Select your USB drive as the target
  5. Click "Flash" or "Write" and wait for completion

Mac Users:

  1. Download Etcher from balena.io/etcher
  2. Insert your USB drive
  3. Open Etcher and select your Elementary OS ISO
  4. Select the USB drive
  5. Click "Flash" and wait for completion

Linux Users:

  1. Use the terminal command: sudo dd if=elementaryos.iso of=/dev/sdX bs=4M
  2. Replace sdX with your USB drive identifier (check with lsblk)
  3. Wait for the operation to complete

Installation Process

Step 1: Boot from USB

  1. Insert the bootable USB drive into your computer
  2. Restart your computer
  3. Press the boot key during startup (usually F12, ESC, or DEL)
  4. Select the USB drive from the boot menu

Step 2: Elementary OS Live Environment

  1. Elementary OS will load with a beautiful interface
  2. You'll see the desktop with an "Install Elementary OS" option
  3. Click "Install Elementary OS" to begin installation

Step 3: Language and Keyboard

  1. Select your language
  2. Select your keyboard layout
  3. Click "Next"

Step 4: Try or Install

  1. Click "Install Elementary OS"
  2. Choose your installation location
  3. Agree to the terms and conditions

Step 5: User Account

  1. Enter your full name
  2. Create a username
  3. Create a strong password
  4. Choose whether to encrypt your home directory (optional)
  5. Click "Next"

Step 6: Disk Configuration

  1. Choose "Erase disk and install Elementary OS" for fresh installation
  2. Or choose "Something Else" for manual partitioning
  3. Review the installation summary
  4. Click "Install" to begin

Step 7: Installation Complete

  1. Wait for the installation to complete (usually 10-15 minutes)
  2. You'll see a completion message
  3. Click "Restart Now" to reboot into your new Elementary OS system
  4. Remove the USB drive when prompted

First Boot and Initial Setup

After restarting, you'll see the Elementary OS login screen:

  1. Enter your username and password
  2. Complete the welcome setup (optional)
  3. Explore the beautiful Pantheon desktop environment
  4. You're now ready to use Elementary OS!

Essential First Steps

1. Update Your System

Keep Elementary OS secure and up-to-date:

  1. Open "Settings"
  2. Navigate to "System"
  3. Click "Updates" to check for system updates
  4. Install available updates

2. Customize Your Desktop

Elementary OS is highly customizable through the Settings application:

  1. Open "Settings"
  2. Explore "Desktop", "Dock", and "Appearance" options
  3. Customize colors, icons, and window preferences

3. Install Applications

Open the "AppCenter" to discover and install applications for your needs.

Troubleshooting

Computer Won't Boot from USB

  • Check that you're pressing the correct boot key
  • Verify the USB drive was properly created
  • Try disabling Secure Boot in BIOS
  • Try a different USB port

Installation Freezes

  • Don't interrupt; allow it to complete
  • Try disabling Secure Boot in BIOS
  • Ensure you have at least 20GB of free space

Screen Flickering or Display Issues

  • Open "Settings" and check "Display" settings
  • Try changing the refresh rate
  • Update graphics drivers through Software Updates

Need Help?

  • Elementary OS Documentation: elementary.io/docs
  • Community Forum: elementaryos.stackexchange.com
  • Ask for help in Linux forums and communities

Congratulations!

You've successfully installed Elementary OS. Enjoy the beautiful interface!
